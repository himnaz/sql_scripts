Create user PRBUK\SVC_MMRAFIUAT on database SQLMULIVWSK14\INST80.

This user will be used by the applications SRPMiddleware, CRAProxy and CRAProxyHelper for database connectivity.

The following permissions required are :

* Access to databases SRPCache, ProxyCache and ProxyCacheArchive 
* Execute permission on all stored procedures in SRPCache, ProxyCache and ProxyCacheArchive 
* SELECT, INSERT, UPDATE, DELETE on all tables in ProxyCache and ProxyCacheArchive
